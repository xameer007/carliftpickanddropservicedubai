## Deployment on Vercel

### Method 1: Using Vercel CLI

1. Install Vercel CLI:
```bash
npm install -g vercel
```

2. Navigate to project directory:
```bash
cd carlift-website
```

3. Deploy:
```bash
vercel
```

4. Follow the prompts and your site will be live!

### Method 2: Using Vercel Dashboard

1. Go to https://vercel.com
2. Click "New Project"
3. Import your repository or drag & drop the folder
4. Vercel will automatically detect and deploy

### Method 3: GitHub Integration

1. Push this folder to a GitHub repository
2. Connect your GitHub to Vercel
3. Import the repository
4. Automatic deployments on every push!

## Customization Guide

### Update Contact Information

Replace all instances of:
- Phone: `+971 50 123 4567`
- Email: `info@dubaicarlift.ae`
- WhatsApp: Update the WhatsApp link numbers

### Update Pricing

Edit the pricing section in `pricing.html` and update route prices in the pricing table.

### Add Routes

Add new route cards in `routes.html` following the existing structure.

### Modify Services

Edit service cards in `services.html` to add/remove services.

### Change Colors

In `styles.css`, update the CSS variables:
```css
:root {
    --primary-color: #2563eb;
    --secondary-color: #0ea5e9;
    /* etc */
}
```
